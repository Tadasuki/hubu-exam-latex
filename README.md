# 湖北大学课程考试试题纸 LaTeX 模板

本项目包含了一个完整的**湖北大学课程考试试卷 LaTeX 模板**。为了便于修改和维护，项目已将样式、配置与内容进行模块化拆分。

## 目录结构说明

```text
环境工程原理试卷/
├── font/                  # 相关字体文件
├── 1.png                  # 试卷顶部的湖北大学校名标志图片
├── exam_configs.tex       # 考试基本信息配置，包含学年、学期、课程名、学院、专业年级等
├── exam_content.tex       # 试卷的具体题目内容，方便在此直接编辑题干和空格
├── exam_style.tex         # 排版与样式配置文件，定义页面边距、密封线、字体及排版命令
├── exam_template.pdf      # 编译生成的 PDF 试卷预览成品
└── exam_template.tex      # 主入口文件，负责引入各模块并进行最终编译
```

---

## 编译

请确保系统已安装`\font`文件夹内相应的中文字体（如华文中宋等），并使用 **`XeLaTeX`** 编译器进行编译：
```bash
xelatex exam_template.tex
```

参考资料：[考试相关下载 - 湖北大学本科生院](https://jwc.hubu.edu.cn/info/1038/1257.htm)
